# MOC Automation – Workflow Solution

##  Overview
This project represents an automated Management of Change (MOC) process implemented using Maestro, Data Fabric, and UiPath integration.

##  Key Features
- MOC request submission through digital forms
- Automatic ID generation
- Multi-department approval workflow
- Validation of approval status
- Final approval by management
- Exporting approved data to Excel

##  Process Flow
1. User submits MOC request form
2. System generates unique request ID
3. Request is reviewed by multiple departments
4. System validates all approvals
5. Final approval is granted
6. Data is exported to Excel report

##  Technologies Used
- Maestro Applications
- Data Fabric
- UiPath Studio
- Excel Automation
